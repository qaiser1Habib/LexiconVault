## Introduction

**MessageMate** is a chatbot application built on **Vite v4.4.8** and **Node.js v18.17.0**. It provides a user-friendly interface to interact with a chatbot for messaging and automated responses.  

## Project Structure

The repository is organized into the following directories:

- `website`: Contains the frontend chatbot interface.  
- `backend`: Contains the backend server code for message handling, storage, and chatbot logic.  

## Installation

1. **Clone the Repository:**  

   ```sh
   git clone https://github.com/qaiser1Habib/MessageMate
   ```

2. **Navigate to Project Root:**  

   ```sh
   cd MessageMate
   ```

3. **Install Dependencies:**  

   ```sh
   npm install
   ```


## Ignored Folders

The following folders are excluded from version control via `.gitignore`:

- `/node_modules`: Dependencies.  
- `/website/public/assets/`: Frontend assets.  
- `/backend/assets/`: Backend assets.
